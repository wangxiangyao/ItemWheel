# ModSettingAPI 集成说明

## 📦 文件下载

ModSettingAPI.cs 文件 **不会** 自动包含在此项目中，需要手动下载。

### 下载地址

从 ModSetting GitHub 仓库下载最新版本的 ModSettingAPI.cs：

**GitHub URL**: https://github.com/duckov/ModSetting

### 下载步骤

1. 访问 https://github.com/duckov/ModSetting
2. 找到 `ModSettingAPI.cs` 文件
3. 点击 "Download" 或 "Raw" 按钮下载
4. 将文件保存到本项目的 `Integration\` 文件夹中：
   ```
   D:\02_projects\Mod\Duckov\ItemWheel\Integration\ModSettingAPI.cs
   ```

## 🎯 为什么需要手动下载

1. **版权和许可证**：遵循 ModSetting 项目的开源协议
2. **版本控制**：始终使用最新版本的 API
3. **更新及时**：ModSetting 更新后，可以立即获取最新版本

## ✅ 验证安装

下载完成后，重新编译项目：

```bash
cd D:\02_projects\Mod\Duckov\ItemWheel
dotnet build -c Debug
```

如果编译成功，说明 ModSettingAPI.cs 已正确放置。

## 🎮 游戏中使用

1. 订阅并启用 ModSetting Mod（创意工坊）
2. 安装本 Mod（ItemWheel）
3. 在游戏中按 ESC → Mod设置 → 找到 ItemWheel
4. 配置各项设置
